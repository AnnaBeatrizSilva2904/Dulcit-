Criação do projeto:

 Intalações de dependências:
 - cd .\backend\ (fazer esse comando no terminal do VScode)
 - npm install (fazer esse comando no terminal do VScode)

 Banco de Dados:
 - Abrir o XAMPP e ativar o MySQL
 - copiar o banco de dados no MySQL Workbench
 - executar o banco de dados

 Terminal:
 - abrir o terminal no VScode
 - entrar na pasta "backend"
 - digitar "node servidor.js" no terminal

 Extensões a serem instaladas:
 - instalar a extensão "Live server"
 - entrar na pasta frontend e abrir o index.html
 - apertar no botão "Go Live" para abrir o projeto no navegador
